\texttt{pseudocode.sty} defines a language with listings package for writing algorithms in a natural way (in english or in french).

\bigskip
The following \LaTeX{} code:
\begin{tex}
\begin{pseudo}
Function integer greatestCommonDivisor(integer a, integer b)
begin
    while (a <> b) do 
        if a > b
           a := a - b
        else
           b <- b - a
        endif
    endwhile
    return a
end
\end{pseudo}
\end{tex}

produces:

\begin{pseudo}
Function integer greatestCommonDivisor(integer a, integer b)
begin
    while (a <> b) do 
        if a > b
           a := a - b
        else
           b <- b - a
        endif
    endwhile
    return a
end
\end{pseudo}
